# فاتورتي - Facturati

نظام تسيير فواتير احترافي (موقع إلكتروني) مصمم خصيصاً لأصحاب المشاريع الصغيرة والمتوسطة في الجزائر.

## المميزات
- واجهة مستخدم احترافية تدعم اللغة العربية والفرنسية.
- صفحة هبوط (Landing Page) جذابة للتعريف بالخدمة.
- إنشاء الفواتير وتعديلها في ثوانٍ معدودة.
- حفظ تلقائي للفواتير في متصفح المستخدم (LocalStorage).
- طباعة الفواتير بمقاس A4 احترافي.
- متوافق تماماً مع الهواتف الذكية والأجهزة اللوحية.

## التقنيات المستخدمة
- React.js
- Tailwind CSS
- Lucide React (Icons)
- React-to-Print

## التشغيل محلياً

1. قم بتحميل المشروع:
```bash
git clone <url-repository>
```

2. قم بتثبيت المكتبات:
```bash
npm install
```

3. ابدأ تشغيل المشروع:
```bash
npm run dev
```
